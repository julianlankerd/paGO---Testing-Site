Custom Shipping Rules

The custom shipper plugin allows you to set your own shipping rules which can be based on weight if required. Consider the following scenario, you want to charge $2.33 for items weighing under 10kg, $5.87 for items over 10kg and under 20kg and a set price of $10.76 for the rest, ie items over 20kg. You would thus create 3 rules with the same name:

Name Weight Rule Price
Ground <10 2.33
Ground >=10 && <=20 5.87
Ground 10.76

It should be noted that order is important in the above. The system will check each rule from the top down and when it finds a match that rule will be used and no other rule WITH THE SAME NAME will be considered. It should also be noted that a rule with a blank weight rule parameter will be accepted automatically so these can be used as a "catch all" type parameter and should be added at the bottom of the list for a group of rules with the same name.
